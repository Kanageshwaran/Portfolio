# Portfolio

A Supabase-backed portfolio concept for showcasing academic work with growth in mind.

## What's included
- High-level design blueprint for a dynamic site that pulls subjects, courses, and assignments from Supabase.
- Supabase data model recommendations for subjects, courses, assignments, and activities.
- Guidance for Times New Roman typography, responsive layouts, and back-button navigation across pages.

See [`docs/design.md`](docs/design.md) for detailed information architecture, component guidelines, and Figma file structure.
